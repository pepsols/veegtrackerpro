# Dashboard

Eerste statische HTML/CSS/JS basis voor het beheerdashboard.

Bestanden:

- `index.html`: overzichtspagina
- `login.html`: eerste statische loginpagina
- `routes.html`: route-overzichtspagina
- `style.css`: layout en styling
- `app.js`: voorbeelddata en eenvoudige rendering
- `server.js`: kleine lokale static server voor `localhost`
- `package.json`: startscript voor lokaal testen

Lokaal starten:

1. Open een terminal in `dashboard/`
2. Run `npm start`
3. Open `http://127.0.0.1:4173`

Belangrijk:

- Voor Google-login gebruik je het dashboard het best via `localhost` of een HTTPS-host
- De server opent standaard `login.html` op `/`
